# 🍎 Fruitkha - Fresh & Organic E-Commerce 

[cite_start]**Fruitkha** is a modern, responsive, and clean E-Commerce Web designed for organic fruits and food delivery businesses[cite: 2]. [cite_start]The primary objective of this project is to provide a user-friendly interface where customers can browse fresh products and place orders seamlessly[cite: 3]. [cite_start]It covers the entire workflow from front-end design to back-end contact functionality[cite: 4].

---

## 🚀 Key Highlights

* [cite_start]**Fully Responsive Design:** Works perfectly on all screen sizes, including Mobile, Tablet, and Desktop[cite: 5].
* [cite_start]**Interactive UI:** Features smooth animations, a preloader, and sticky navigation[cite: 6].
* [cite_start]**E-commerce Ready:** Includes a complete flow for Shop, Cart, and Checkout pages[cite: 6].
* [cite_start]**Functional Contact System:** Integrated with a PHP-based mail system including validation[cite: 7].

## 📩 Form Functionality

### **The project utilizes a mail.php file that provides the following features:**

* Token Validation: Ensures secure form submission.
* Data Sanitization: Cleans input data before converting it into an email format.
* Success Feedback: Displays a confirmation message to the user after the message is sent.
